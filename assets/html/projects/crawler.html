Project Crawler

Status: In Development / Unfinished

View on Github

Project Crawler is an attempt at replicating a dungeons and dragons character creator based on the likes of

D&D Beyond & Dungeon Master's Vault

I am an avid D&D Enjoyer and with very few alternatives out there, I wanted to take a shot at making my own character builder.

I have not yet got to the stage of building the main application, but i have used it to learn more about async requests using jquery's ajax features, building up my knowledge of front-end development and server side development by hosting a webserver locally with a database using MariaDB, Apache and Ubuntu Server.
